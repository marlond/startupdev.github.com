Caso o cliente ache que o escopo acordado é muito pouco, ele tem o direito de rescindir o contrato e abortar o desenvolvimento, sem ter que pagar multa. Neste caso, o sinal fica como pagamento pela análise do projeto que é realizada na reunião.

Nós só partimos para o desenvolvimento caso o cliente esteja seguro sobre o que será desenvolvido. Na entrega do projeto é realizado o pagamento dos 80% restantes.
